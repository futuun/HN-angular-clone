##What is this?
Hacker News app made in angular.
Nothing fancy i just wanted to try out this API.

###Tools used:
 - Angular
 - Firebase
 - [Hacker News API](https://github.com/HackerNews/API)
 - TypeScript
 - webpack

##Instalation

```
npm install
```
##Development
```
npm start
open loacalhost:1337 in the browser

```

##Build
```
npm run build
```
